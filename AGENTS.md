# Project Instructions

- If a requested filename, path, command, or technical term looks misspelled or unusual, ask for confirmation before proceeding.
- Pay special attention to English filenames. For example, if a name looks like a typo of a common word, confirm it first.
- Proceed with unusual names only when the user explicitly confirms they want that exact spelling.
- Prefer Hugo overrides in root `layouts/`, `static/`, or `assets/` instead of editing the theme source directly.
- Do not commit Obsidian settings or private local writing templates.
